# Desert Rage [EN]
The game that you're all were waiting for...

Guide to start playing:
1. The first thing that you need to do is download "Desert Rage.exe"-file
2. If your PC has .NET Core 3.1 and SSDT tools then skip until step 3
- Download and go to the "Components"
- Launch "SSD-Setup-RUS.exe"
- Accept all
- Launch "windowdesktop-runtime-3.1.12-win-<your Windows version>.exe"
- Accept all
3. Then open "Desert Rage.exe" application
4. Enjoy yourselves
